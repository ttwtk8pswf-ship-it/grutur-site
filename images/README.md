# IMAGENS DO SITE GRUTUR

Esta pasta contém as imagens utilizadas no site da Grutur - Ecoturismo, Lda.

## Imagens Necessárias:

1. **logo-grutur.png** - Logo da empresa Grutur (com árvores verdes)
2. **tramontina-restauracao.jpg** - Imagem dos produtos de restauração e hotelaria (Tramontina)
3. **supertaxi.jpg** - Imagem do serviço de táxi por aplicativo (SuperTaxi)
4. **inuka-cosmeticos.jpg** - Imagem dos produtos cosméticos (INUKA)

## Instruções:

As imagens devem ser colocadas nesta pasta (`images/`) com os nomes exatos listados acima.

Essas imagens já existem no repositório GitHub da Grutur e devem ser copiadas para cá.

## Formatos Recomendados:

- **Logo**: PNG com fundo transparente (recomendado: 200x200px ou maior)
- **Fotos de serviços**: JPG de alta qualidade (recomendado: mínimo 800x600px)

## Otimização:

Para melhor performance do site, é recomendado:
- Comprimir as imagens JPG (qualidade 80-85%)
- Usar PNG otimizado para o logo
- Dimensões máximas: 1920x1080px para fotos de serviços
